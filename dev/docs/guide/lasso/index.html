Lasso

The Lasso (Least Absolute Shrinkage and Selection Operator) is a linear regression model with an ℓ1\ell_1ℓ1​ penalty that induces sparsity — it drives some coefficients exactly to zero, effectively performing feature selection.

Objective Function​

min⁡w12n∥Xw−y∥22+α∥w∥1\min_w \frac{1}{2n} \|Xw - y\|_2^2 + \alpha \|w\|_1wmin​2n1​∥Xw−y∥22​+α∥w∥1​

where nnn is the number of samples and α≥0\alpha \ge 0α≥0 controls the regularization strength. This formulation matches scikit-learn's Lasso.

Coordinate Descent Algorithm​

Skigen solves the Lasso via coordinate descent with soft-thresholding. At each iteration, the algorithm optimizes one coefficient wjw_jwj​ at a time while holding all others fixed.

Define the partial residual correlation:

ρj=Xj⊤ ⁣(y−X∖j w∖j)\rho_j = X_j^\top\!\left(y - X_{\setminus j}\, w_{\setminus j}\right)ρj​=Xj⊤​(y−X∖j​w∖j​)

where X∖jX_{\setminus j}X∖j​ denotes all columns except jjj. The closed-form update for coordinate jjj is:

wj←S(ρj,  nα)∥Xj∥22w_j \leftarrow \frac{S(\rho_j,\; n\alpha)}{\|X_j\|_2^2}wj​←∥Xj​∥22​S(ρj​,nα)​

where S(z,γ)=sign⁡(z)max⁡(∣z∣−γ, 0)S(z, \gamma) = \operatorname{sign}(z)\max(|z| - \gamma,\, 0)S(z,γ)=sign(z)max(∣z∣−γ,0) is the soft-thresholding operator.

In practice, ρj\rho_jρj​ is computed efficiently using the running residual r=y−Xwr = y - Xwr=y−Xw:

ρj=Xj⊤r+∥Xj∥22 wjold\rho_j = X_j^\top r + \|X_j\|_2^2 \, w_j^{\text{old}}ρj​=Xj⊤​r+∥Xj​∥22​wjold​

The algorithm cycles over all features until convergence (maximum coefficient change <tol< \texttt{tol}<tol) or max_iter iterations.

Sparsity and Feature Selection​

The ℓ1\ell_1ℓ1​ penalty produces sparse solutions: at convergence, any coefficient whose partial correlation ∣ρj∣|\rho_j|∣ρj​∣ does not exceed nαn\alphanα is set exactly to zero. This makes Lasso a natural tool for feature selection in high-dimensional settings.

When to Use​

  • Feature selection: When you expect many features to be irrelevant.
  • Sparse signals: When the true underlying model involves only a few non-zero coefficients.
  • For correlated features, consider ElasticNet, which combines ℓ1\ell_1ℓ1​ and ℓ2\ell_2ℓ2​ penalties for stability.
