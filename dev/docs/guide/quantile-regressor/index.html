QuantileRegressor

QuantileRegressor fits a linear model that predicts a conditional quantile of the target rather than its mean. Where ordinary least squares minimises squared error (and so estimates the conditional mean), quantile regression minimises the asymmetric pinball loss and estimates a chosen quantile τ∈(0,1)\tau \in (0, 1)τ∈(0,1) — the median for τ=0.5\tau = 0.5τ=0.5, the 90th percentile for τ=0.9\tau = 0.9τ=0.9, and so on.

Objective Function​

For quantile τ\tauτ and L1 penalty strength α\alphaα:

min⁡w,b  1n∑i=1nρτ ⁣(yi−xi⊤w−b)+α∥w∥1,\min_{w, b}\; \frac{1}{n}\sum_{i=1}^{n} \rho_\tau\!\big(y_i - x_i^\top w - b\big) + \alpha \lVert w \rVert_1,w,bmin​n1​i=1∑n​ρτ​(yi​−xi⊤​w−b)+α∥w∥1​,

where the pinball loss is

ρτ(r)=max⁡(τr,  (τ−1)r)={τ rr≥0,(τ−1) rr<0.\rho_\tau(r) = \max(\tau r,\; (\tau - 1) r) = \begin{cases} \tau\, r & r \ge 0,\\ (\tau - 1)\, r & r < 0. \end{cases}ρτ​(r)=max(τr,(τ−1)r)={τr(τ−1)r​r≥0,r<0.​

The asymmetry of ρτ\rho_\tauρτ​ penalises under- and over-prediction differently, which is what pins the fit to the requested quantile.

Linear Programming Formulation​

The objective is piecewise-linear, so it is solved exactly as a linear program. Splitting each residual into positive and negative parts ui,vi≥0u_i, v_i \ge 0ui​,vi​≥0 and each coefficient into wj+,wj−≥0w^+_j, w^-_j \ge 0wj+​,wj−​≥0 gives the standard-form LP

min⁡z≥0c⊤zs.t.X(w+−w−)+(b+−b−)+u−v=y,\min_{z \ge 0} c^\top z \quad\text{s.t.}\quad X(w^+ - w^-) + (b^+ - b^-) + u - v = y,z≥0min​c⊤zs.t.X(w+−w−)+(b+−b−)+u−v=y,

with pinball weights τ\tauτ on uuu and 1−τ1 - \tau1−τ on vvv, and the L1 weight αn\alpha nαn on w++w−w^+ + w^-w++w−. Skigen solves this with a native primal-dual interior-point method (Mehrotra predictor-corrector), so no external LP backend is required and Eigen remains the only dependency.

Example​

#include <Skigen/LinearModel>

for (double q : {0.1, 0.5, 0.9}) {
    Skigen::QuantileRegressor<double> model(q, /*alpha=*/0.0);
    model.fit(X, y);
    // model.coef(), model.intercept() now describe the q-th quantile line.
}

Parity Scope​

Mirrors sklearn.linear_model.QuantileRegressor for the quantile, alpha, and fit_intercept parameters and the coef_ / intercept_ / n_iter_ attributes. Only the native interior-point solver is provided; sklearn's "highs", "highs-ds", "highs-ipm", and "revised simplex" solver names are not selectable (passing them raises). solver_options and sample_weight are not honoured.
